# Automated Authentication Flow

This repository contains automated scripts for handling user registration and login workflows securely and efficiently using **Python** and **Playwright**.

##  Features
- **Automated Signup:** Handles dynamic user registration via `signup_automation.py`.
- **Automating Sign-in:** Validates user sessions and login flows via `signin_automation.py`.
- **Captcha Bypass:** Integrated with `2Captcha` using environment variables for seamless solver mechanics.
- **Proxy Support:** Rotates through proxy configurations to ensure smooth background network activity.

##  Tech Stack
- **Language:** Python
- **Automation Framework:** Playwright (Python-built Core)
- **Environment Management:** Python-dotenv

*Note: Sensitive configuration tokens and local `.env` setups are excluded from this repository for security compliance.*
